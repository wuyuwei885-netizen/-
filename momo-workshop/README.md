# 极速表情包工坊

一个可直接双击运行的纯前端表情包生成器，使用 HTML + CSS + JavaScript + Canvas 实现。支持模板、导入图片、文字、贴纸、裁剪、滤镜、调节、自定义尺寸、图层管理、撤回重做、生成成品预览和本地作品浏览。

## 本次修复重点

- 修复裁剪模式下裁剪框内部显示黑色的问题：现在裁剪遮罩只覆盖框外区域，框内保留原图预览。
- 修复应用裁剪时把半透明遮罩一起裁进去的问题：裁剪会使用干净离屏 Canvas 合成，不会把黑色遮罩导出。
- 增强图层功能：图层列表可点击选择，支持删除、显隐、锁定、上移、下移、置顶。
- 新增图片镜像：支持背景水平镜像、背景垂直镜像。
- 新增图层镜像：选中文字或贴纸后，可水平/垂直镜像当前图层。
- 表情包模板也能作为贴纸粘贴到画布上。

## 本地运行

1. 解压压缩包。
2. 进入 `meme-workshop` 文件夹。
3. 双击 `index.html`。
4. 浏览器打开后即可使用。

## 核心功能

- 内置 8 个表情包模板。
- 内置小红书风格贴纸，并支持模板贴纸化。
- 支持上传 jpg/png/webp 图片。
- 支持背景 cover/contain/stretch/center 适配。
- 支持背景水平/垂直镜像。
- 支持文字图层：拖拽、缩放、旋转、颜色、描边、阴影。
- 支持贴纸图层：拖拽、缩放、旋转、复制、镜像、删除。
- 支持裁剪比例：自由、1:1、3:4、4:3、9:16、16:9。
- 支持滤镜和基础调节。
- 支持输出尺寸自定义。
- 支持撤回/重做。
- 支持 PNG/JPG 生成、下载、复制到剪贴板。
- 支持作品浏览，本地最多保存 12 张。

## 部署到 GitHub Pages

1. 新建 GitHub 仓库。
2. 上传 `index.html`、`style.css`、`script.js`、`README.md`。
3. 进入仓库 Settings → Pages。
4. Source 选择 main 分支，目录选择 root。
5. 保存后等待 GitHub Pages 生成链接。

## 项目亮点

该项目不是静态网页，而是一个轻量 Canvas 表情包编辑器。它包含图层管理、贴纸粘贴、裁剪、滤镜、调节、镜像、撤回重做和高清导出，适合课程展示、前端项目汇报和 GitHub Pages 部署。
